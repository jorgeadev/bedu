# 🍃 Matcha

### Build your blog. Build your business.

**Matcha** is a premium ecommerce blogging platform designed to help brands instantly publish articles, drive more traffic, grow their email lists, and measure their blog’s impact on sales.

---

## 🚀 Features

- **⚡ Instant Publishing**: Fill your blog with engaging articles in minutes, not hours.
- **📈 Traffic Generation**: Attract and engage more website visitors with high-quality content.
- **📧 Email Capture**: Convert traffic into loyal subscribers with powerful lead-gen tools.
- **📊 Performance Insights**: See clearly which content delivers the most ROI.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop.
- **🎨 SASS Powered**: Beautifully crafted styles using modern SCSS architecture.

## 🏁 Getting Started

### Prerequisites

This project uses **pnpm** for package management. Ensure you have it installed globally:

```bashw
npm install -g pnpm
```

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/jorgeadev/bedu.git
   ```

2. **Install dependencies**:

   ```bash
   pnpm install
   ```

3. **Running the project**:
   Since it's a static site, you can open `index.html` directly in your browser or use a live server extension.
